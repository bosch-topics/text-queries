Access Request Instructions

Step 1: Check Access in OneIDM

Kindly check the access in OneIDM for default access in PH1011 & PXG011 IT applications. For checking the access rights, kindly click the link: https://oneidm.bosch.com/oneidm/html/rbwebshop/#/start/itshop/VI_Start

Step 2: Extend or Revoke Access

Click Request → Extend or Revoke → Click the filter option in IT application → Search for PH1011, PXG011 & Portal IT application.

Step 3: Filter and Check Default Access

Apply filter in Services and check for the default access assigned. Check for the default access:

  • In PH1 - X:H0SY_XXXX_ALL_M_MINUSER_HR, X:H0HR_INXX_MA_ESS
  • In PXG - X:0GFI_FIORI_SSF_MYACC_PDX, X:0GXX_FIORI_ALL_MINUSER, X:0GHR_FIORI_HCM_IN_HPE_PH1, X:0GHR_FIORI_HCM_IN_ESS_PH1
  • In Portal - CN=org_every_rbei_associate, OU=portal, OU=groups, OU=esi, DC=wam-bosch, DC=com, CN=rb_myhr_associate_in, OU=portal, OU=groups, OU=esi, DC=wam-bosch, DC=com, CN=rb_fi_myaccounting_self_service_new_ui, OU=portal, OU=groups, OU=esi, DC=wam-bosch, DC=com, CN=rb_fi_myaccounting_self_service, OU=portal, OU=groups, OU=esi, DC=wam-bosch, DC=com, CN=rb_flp_hr_core_users, OU=portal, OU=groups, OU=esi, DC=wam-bosch, DC=com

If the access rights are not assigned, kindly raise a ticket in OneIDM Support request using the link below. If the access is already assigned, check for account activation.

Step 4: Account Activation

Click Accounts Tab in OneIDM → Choose My accounts → Check for IT Application PH1011 → Check for the exclamatory symbol in the Status column → Choose the dropdown → Click on Activate account → Enter the valid reason → Click Yes.

Step 5: Request Number Generation

A combined request number will be generated after the activation.

Step 6: Check Request Status

To check the status in OneIDM, click Request → Find Request → Enter CRN in "Search by Request Id" and check the Status column. Initially, the status will be updated as "New", but once the status changes to "Assigned", kindly try to access the application.

If Access is Still Unavailable:

If you are unable to access the application, kindly register your request/query through the link: https://boschindia.service-now.com/sp?id=sc_cat_item&sys_id=ed35579747e1ce10750cc6de116d4319

If prompted, “Login with SSO” using NTID, or you can also create a ticket by navigating from the Employee Self Service portal → OneIDM & BGSW IT Applications Support → OneIDM Support Request.

Step 1: Create Ticket

Click "Request for" and select "Myself or Others".

Step 2: Beneficiary NT ID

Enter the "Beneficiary NT ID".

Step 3: Select Application

Select Application as "oneIDM".

Step 4: Select Module

Select Module as "PEI011".

Step 5: Business Justification for Access

Enter the "Business reason" in the "Business justification for access".

Step 6: Describe the Access Issue

Enter the details about access or issue in "Describe access right / Tcode / Issue".

Step 7: Attach Screenshot

Add any error screenshot as an attachment.

Step 8: Submit the Request

Click "Submit", and a ServiceNow ticket will be created.

Step 9: Email Notification

An email notification will be triggered when the ticket is created, and the ticket status can be checked in the ServiceNow portal. Additionally, email notifications will be sent for every update or change to the ticket.

Step 10: Check Ticket Status

To check the status of the created ticket in the ServiceNow portal, kindly click the "Request" option. The created ticket number will be listed there, and the user can check the ticket status.
